# 说书体风格指南

## 总原则

听得懂是第一位，古典味是第二位。古典味来自节奏、句式和说书人姿态，而不是生僻字词。

## 句式工具箱

说书人姿态（每 3-4 句最多用一个，轮换使用）：

- 开场拉人：列位看官 / 各位且听
- 设问勾连：你道为何 / 你道这人是谁
- 转场：且说 / 话分两头 / 按下不表
- 收尾：欲知后事如何 / 这正是……

节奏手段：

- 三句递进：一句陈述、一句加码、一句反转。
- 对仗短句压节奏：人是好人，事不是好事。
- 关键名词重复制造悬念：那封信，偏偏就是那封信。

## 正反例

**开头**

反例（背景先行）：明朝成化年间，徽州府有一商人，姓程名宰，家中颇有资财，常年在外行商。

正例（钩子先行）：一个大活人，睡到半夜，床底下伸出一只手来。列位看官，这事就出在徽州程家。

**用词**

反例：程宰怫然不悦，拂袖而去。

正例：程宰脸一沉，袖子一甩，走了。

**悬念收尾**

反例（每集雷同）：欲知后事如何，且听下回分解。

正例（轮换）：
- 这只手到底是人是鬼？咱们下集接着说。
- 他万万没想到，救他的人，正是要他命的人。
- 这正是：贪心起处灾星到。下集，灾星进门。

## TTS 安全规则

- 全部数字写汉字：二十两、三更天。
- 多音字易错处改写：「还钱」写「把钱还上」可避免读成 hái。
- 人名首次出现给身份：徽州商人程宰，之后只称程宰。
- 避免书面缩略：不写「程因病卒」，写「程宰得了急病，没熬过去」。

## 道德落点（下集专用）

下集最后 2-3 句收一个落点，从这些角度轮换，不要每集都是因果报应：

- 因果有报
- 贪念误身
- 善念救命
- 世事无常，人心可测
- 留一线余地，日后好相见
