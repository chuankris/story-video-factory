# Output Spec

Use this reference when preparing outputs for Douyin or similar short-video platforms.

## Video Defaults

- Aspect ratio: `9:16`
- Resolution: `1080x1920`
- Frame rate: `25` or `30` fps
- Duration target: depends on content, but MVP comic-video drafts should usually stay concise and retention-focused.
- Audio: clear narration first, background music second.

## Required Delivery Artifacts

For a composed video:

- `output/draft.mp4`
- accepted script
- storyboard
- source images or generated clips
- audio files
- optional quality report

For a publishing pack:

- title options
- cover text options
- caption
- hashtags
- risk notes
- suggested opening frame or cover image

## Quality Checklist

Check:

- first 3 seconds have a hook
- story remains understandable without source text
- subtitles match narration
- no missing image/audio assets
- no obvious image artifacts or modern visual contamination for historical stories
- no audio clipping
- output resolution and orientation are correct
